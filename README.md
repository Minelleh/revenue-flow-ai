# Jude · the website apostle

The agent that built executionsquad.co. Yours to install.

## What Jude is

Jude builds a 2-page lead funnel. Squeeze page captures email. Delivery page hands over the offer. Same shape as executionsquad.co. The bundle ships ready to drag into your project root.

## What Jude is NOT

Read this before you build. The same questions come in every week.

- **Not a multi-page brochureware site.** No Home, About, Services, Team, Contact, Blog. If your client wants a 7-page agency site, Jude is the wrong tool.
- **Not a website-agency-in-a-box.** The point is your first artifact and your first 5 paying clients on your own offer. Path B (selling installs to local businesses for $3,000) is a real option, not the product.
- **Not the $10K AIOS install consultants sell.** Squad is an AI operating system installed into your own folder, not into a client's company.
- **Not a long-term moat.** The 3D angle is a 6 to 12 month novelty window. Use Jude to ship fast, then move to Phase 3 outreach, Phase 4 content, Phase 5 ads.

## Running Jude across multiple clients

If you are shipping for several local businesses (Path B), the structure is:

- One project folder per client. Same `.claude/` inside each.
- One `business.md` per client. Different offer, different voice, different output.
- Leads land in your own Supabase. Split by table or by project per client, your call.

The bundle is reusable. The work is the `business.md`.

## Install in 30 seconds

> **Heads up.** The `.claude/` folder is hidden by default in Mac Finder and Windows Explorer. The fastest fix: open this unzipped folder in your code editor (Claude Code, Antigravity, Cursor, VS Code) and press `Cmd + Shift + E` (Mac) or `Ctrl + Shift + E` (Windows) to open the Explorer panel. Editors show hidden folders. The `.claude/` folder will be right there. If you prefer Finder, hit `Cmd + Shift + .` to reveal hidden folders.

1. Open the folder where your project lives. If you do not have one yet, make a new empty folder. Call it whatever your business is.
2. Drag the `.claude/` folder from this download into your project root.

That is it. Jude lives inside your project now.

## Build your first site

**Step 1.** Write your `business.md` at the project root.

If you do not have one yet, watch the John video at executionsquad.co. He is the self-diagnosis apostle. He interviews you for thirty minutes and writes the file from your head. Offer, audience, voice, strategy, brand color, typography, aesthetic descriptor.

**Step 2.** Open Claude Code in the project folder. Paste this prompt:

```
Jude, read business.md and start the website build.
Walk the thirteen phases in /jude-website. Pause for review at each phase.
```

**Step 3.** Approve or fix at every phase. Funnel, copy, VSL script, Higgsfield prompts, 3D hero, components, deploy, anti-slop. Thirteen reviews. You stay the manager. Jude reports.

By the end, you have a deployed two-page funnel with Supabase lead capture, Resend welcome email, and a 3D hero. Same craft level as executionsquad.co. Your accent color, your typography, your aesthetic. Your domain. Your code.
